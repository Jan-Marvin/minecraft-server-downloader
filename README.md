# Minecraft server downloader

A simple tool to download the latest server jar

## Usage
```console
minecraft-server-downloader [-h] [-l] [-o <name>] [-v]
```

## Options
```console
-h, --help 		Print this help
-l			Enable curl verbose information
-o			Set output name
-v, -V, --version	Print version
```
